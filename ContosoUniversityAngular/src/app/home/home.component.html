Contoso University - Angular 4.x Campus
Welcome to Contoso University's Angular 4.x Campus

Contoso University Angular 4.x Campus is a sample application that demonstrates how to use Entity Framework Core 1.1.2 in an ASP.NET Core 1.1.3 Web API that uses Angular 4 and Progress Kendo UI for Angular 2+ for the user interface.
